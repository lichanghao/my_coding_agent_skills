Coding agent skills for automatically generating music recommendation posts on social media.
Author: Changhao Li

Installation (Gemini CLI): `gemini skills install [path to music-curator skill folder] --scope workspace`

See the documentation of other coding agents for skill installation.

Prompt example: "Use music-curator skill to prepare a package of material for a recommended R&B music."

The coding agent will generate idea of music recommendation, search online and download the video, automatically clip the video to ~30s, and write a short caption.
